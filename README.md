## Getting Started

### Prerequisites

Make sure you have the following installed:

- Java 11 or higher  
- Maven  
- Node.js and npm  
- Angular CLI  
- PostgreSQL  
- Postman (or any REST client)

---

## Launch frontend
```
ng serve
```

The app will be available at:
```
http://localhost:4200
```
